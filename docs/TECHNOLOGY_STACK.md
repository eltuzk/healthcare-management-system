# Danh Sách Công Nghệ Sử Dụng Trong Dự Án (Healthcare Management System)

---

## I. FRONTEND

* **Ngôn ngữ lập trình chính**: TypeScript
* **Thư viện UI chính**: React 19
* **Công cụ đóng gói & Build tool**: Vite
* **Thiết kế & Giao diện (Styling)**: TailwindCSS
* **Thư viện hiệu ứng chuyển động**: Framer Motion
* **Thư viện cuộn mượt (Smooth Scrolling)**: Lenis Scroll
* **Thư viện định tuyến & Điều hướng**: React Router DOM (v7)
* **Thư viện kết nối API**: Axios
* **Bộ biểu tượng (Icons)**: Lucide React
* **Xác thực mạng xã hội**: React OAuth Google
* **Thư viện thông báo nhanh**: React Hot Toast

---

## II. BACKEND

* **Ngôn ngữ lập trình chính**: Java 21
* **Framework chính**: Spring Boot (v4.0.3)
* **Hệ quản trị cơ sở dữ liệu (Database)**: Oracle Database (XE)
* **Driver kết nối dữ liệu**: Oracle JDBC Driver (`ojdbc11`)
* **Lớp ánh xạ thực thể (ORM)**: Spring Data JPA (Hibernate)
* **Xử lý logic mức Database**: PL/SQL (Stored Procedures, Functions, Triggers)
* **Quản lý phiên bản Database**: Flyway Migration
* **Cơ chế xác thực an toàn**: Spring Security
* **Tạo và xác thực Token**: JJWT (Java JWT v0.13.0)
* **Tích hợp cổng thanh toán**: SePay Webhook (tự động nhận chuyển khoản ngân hàng)
* **Xác thực SSO**: Google API Client (OAuth2)
* **Dịch vụ gửi Email tự động**: Spring Boot Mail (Gmail SMTP)
* **Xử lý và xuất nhập file Excel**: Apache POI (poi-ooxml)
* **Tài liệu hóa API trực quan**: Springdoc OpenAPI UI (Swagger UI)
* **Thư viện tối giản code boilerplate**: Lombok
* **Công cụ quản lý thư viện**: Maven
